# Angular Material Themes

To view the final version of the app:

```
npm install

npm start
```

Checkout these branches to follow along with the presentation:

- `01-start`
- `02-custom-theme`
- `03-custom-theme-hex-codes`
- `04-custom-fonts`
- `05-dark-theme`
- `06-dark-theme-switcher`
